ISI export format

The databases in Web of Science offers an export format. The format is used and recognized by most reference management software, and is used by HistCite as well. This page describes the format, the meaning of the fields, and the development of the format.

Some field codes seems to have been added over the years - this document tries to find out when the specific field codes have been added to the format, and to describe when they are used.

The basic export format is a plain text-file, with the US-ASCII character set. There's a word-wrap at 78 characters, line endings are LF.

Tag Meaning Notes
FN Start of file Always "ISI Export Format"
VR Version of file Always (?) "1.0"
PT Publication Type One of the publication types in WoS. Denotes start of record.
AU Authors Only shows surname + first character of first name + middle names
AF Authors, full Shows full name of authors
TI Title of the record
SO Source of the record Always in CAPS. The Master Journal List should contain this title.
AB Abstract Beginning of Abstract-field
PD Publication Date Can be date, or month, or range of months. Not a controlled field.
PY Publication Year The publication year can differ from the year the record was entered into WoS. See Technical Limits
VL Volume
IS Issue
BP First printed page
EP Last printed page
UT Unique Tag An WoS-specific tag. Non-searchable through WoS.
ER End Record Denotes the end of a record
EF End of File
